# OpenSzkołaSite
Prosta strona szkoły<br>
Uruchamianie:
1. Przejrzyj wszystkie pliki i pozmieniaj wartości w {{ uzupełnij }} i zmień nazwę szkoły
2. Uruchom serwer PHP
```bash
php -S localhost:8080
```
